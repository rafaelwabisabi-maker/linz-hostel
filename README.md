# Linz Hostel

Digital infrastructure for a community-driven hostel project in Linz, Austria.

## About

A platform for a new kind of hostel — combining affordable accommodation with community programming, cultural events, and local partnerships. The application handles booking flows, event showcasing, and partner integration. Built to support urban neighborhood revitalization through hospitality innovation.

## Tech Stack

- Next.js 16 / React 19
- TypeScript
- Tailwind CSS
- Radix UI
- Framer Motion
- Recharts (data visualization)
- Vercel AI SDK
