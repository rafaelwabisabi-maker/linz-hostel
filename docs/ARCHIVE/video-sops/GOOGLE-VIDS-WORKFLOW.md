## HOW THIS WORKS (30 seconds)

1. **Rafael/Daniel send me** raw project docs, a partner name, or a story angle
2. **I output** a storyboard table in Google Vids-ready format
3. **You go to** [vids.new](https://vids.new) → click **Storyboard** → paste the table
4. Vids generates scenes from the prompts. Swap clips as needed with Veo sidebar.

Two paths depending on what you have:

| If you have... | Use this path |
|---|---|
| Only text/story | **Storyboard Pipeline** → paste my table into vids.new |
| Existing slides/deck | **Slide-to-Video** → import to Vids → Convert Slides (Speaker Notes = voiceover) |

**For KraftWerk partnership videos → always use Storyboard Pipeline.** The stories are narrative, not slide-friendly.

## TIPS FOR USING GOOGLE VIDS

- Go to **vids.new** → select **Storyboard**
- Paste the entire table from above into the prompt box
- Vids will build the timeline, generate AI voiceover, and pull footage
- Use the **Veo 3.1 sidebar** to swap any clip that generates inaccurately
- Adjust **Voiceover vs. Music** levels in the audio panel after generation
- Export 30-sec cut (scenes 1-4 only) for Instagram Reels

**Premium note (Austria/EU):** Veo 3.1 and AI Avatars are Premium features. Keep scripts concise to stay within monthly AI credit limits.
