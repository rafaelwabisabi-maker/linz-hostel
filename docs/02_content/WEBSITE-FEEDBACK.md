# Linz Hostel Website — Design Feedback & Direction

> Collected notes on feel, interaction, and missing elements.

---

## Overall Feel

The website should feel like **opening an architectural art project book — an encyclopaedia**. Authority, seriousness, discovery, curiosity, then sparks of colour over the "human isolation" problem.

**Reference**: The slides should feel like a cosy Canva slideshow presentation — crafted by experienced hand-draws (Adobe engineers before the computer era).

---

## Specific Feedback

### Slide Presentation
- Slides need to be **more prominent and obviously clickable/slideable** — currently buried
- Consider **more slides** — there's more to be talked about. Check the richness of content from the authors.
- **Background**: should smoothly flow (almost imperceptible) — like soft thick cotton/big paper is behind everything

### Buttons & Interaction
- Buttons need **more life, more interaction, curiosity calling to be clicked** (in an elegant way)
- Currently feels too static

### Visual Richness
- **It's missing pictures!** It misses the art elegancy, the visual enchantment
- For important institutions mentioned → always put their **logo** next to it (upload logos)

### Mobile
- Consider proper software integration for phone viewing
- The slides should appear properly on mobile (currently seems like only the AI chat shows)

### Content Depth
- Feed the **entire project material as a database** for having more answers and a more robust narrative for those who want to keep reading
- Use the pitch decks as knowledge base for the chatbot
