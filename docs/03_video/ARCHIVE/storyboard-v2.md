## PRODUCTION NOTES

### Register Map
- **S1–S5:** DREAM register (illustrated — gouache, watercolor, ink)
- **S6–S8:** PROOF register (illustrated but warmer — the model shown)
- **S9:** DATA register (typography-forward, visual breathing)
- **S10:** TRUST register (real footage — the human)
- **S11:** CLOSE (brand identity)

### Style Rules (from PDF Golden Rules)
- ONE concept per image. No transitions inside a frame.
- Brand palette in EVERY prompt: #4A4543 #C8BCB4 #C4997E #E76E50
- 16:9 native. Never crop after generation.
- 20% dead space on all edges for VEO3 camera movement.
- Deep staging: foreground + midground + background.
- No crowds, no detailed faces, no hands holding objects.
- Cousin styles allowed — palette stays identical.
- Audio-first: nothing moves until the voice is timed.

### Script Chapter Reorder
Script v1 chapters are reordered for visual logic:
- Ch 1 (Problem) → S1+S2+S3
- Ch 2 (Vision) → S4+S5
- Ch 4 (Location) → S6 ← moved before Secret Sauce
- Ch 3 (Secret Sauce) → S7 ← moved after Location
- Ch 5 (Ecosystem) → S8
- Ch 6 (Numbers) → S9
- Ch 7+8 (Team+Ask) → S10
- Ch 0 (Title/Close) → S11

**Rationale:** You see the BUILDING (S6), then what happens INSIDE it (S7), then the ECOSYSTEM around it (S8). Outside → Inside → Network.
