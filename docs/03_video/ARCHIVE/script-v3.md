## TAGLINE DECISION

> **Recommendation:** Change the official tagline from "From Isolation to Connection" → **"Nobody stays a stranger."**

**Why it wins:**
- "From Isolation to Connection" = passive, describes the category
- "Nobody stays a stranger" = active, makes a promise, specific to the visitor experience
- Works as a close: the line lands differently the second time (it's no longer a description — it's a result)
- More memorable. More human. Jury/investor repeats it in their head.

Run past Daniel before changing on website/deck.
